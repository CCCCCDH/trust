Trust сохраняет для вас маленький лог событий.

Будь то изменения приложения, использование телефона другими людьми, пропущенные вызовы, - Trust поведает вам что и когда происходило. 

События, включенные на данный момент:
- Запущенные приложения
- Установленные/Удаленные/Перемещенные/Перезапущенные/Очищенные приложения
- Подсоединение/отсоединение источника питания
- Выходы/Загрузки системы
- События Вкл/Выкл дисплея
- События экрана блокировки
- Утрата wi-fi/сотового соединения
- Отсоединенные носители
- Вызовы

Какие РАЗРЕШЕНИЯ нужны Trust:
*PROCESS_OUTGOING_CALLS для учета входящих/исходящих вызовов
*RECEIVE_BOOT_COMPLETED для автозапуска и учета перезагрузок
*READ_PHONE_STATE для учета изменений подсоединений
*WRITE_EXTERNAL_STORAGE для экспорта базы данных
*GET_TASKS для данных приложений на переднем плане
*ACCESS_WIFI_STATE для wifi SSIDs
