Trust зберігає для вас невеличкий лог подій.

Зміни додатка, хтось інший користувався пристроєм, пропущені виклики... Trust розповість вам що й коли трапилось.

На даний час, до подій віднесені:
- Launched apps
- Встановлені/видалені/переміщені/перезапущені/очищені додатки
- Під'єднання/Від'єднання джерела живлення
- Вимикання/Завантаження системи
- Події Увімкн/Вимкн дисплея
- Події екрана блокування
- Втрата wi-fi/стільникового підключення
- Видалені носії
- Виклики

Яких ДОЗВОЛІВ потребує Trust:
*PROCESS_OUTGOING_CALLS для визначення вхідних/вихідних викликів
*RECEIVE_BOOT_COMPLETED для автостарту та визначення перезавантажень
*READ_PHONE_STATE для визначення змін під'єднань
*WRITE_EXTERNAL_STORAGE для експорту бази даних
*GET_TASKS for current foreground apps
*ACCESS_WIFI_STATE for wifi SSIDs
