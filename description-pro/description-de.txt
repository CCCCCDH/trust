Diese App schaltet den Pro Status in der kostenlosen Version von Trust frei.
Sie hat keinerlei Funktion und muss lediglich zusammen mit der kostenlosen Version von Trust installiert sein.

Durch die Installation des Unlockers wird die maximale Zeit von gespeicherten Ereignessen von 24 Stunden auf eine Woche erweitert.
